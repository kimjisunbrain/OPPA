# OPPA

# Repository details
Hello!
This repository contains codes used to collect data for the following paper:

Kim, J. S., Seol, J. M., Lee, C. H., Lee, I., & Lee, S. A. (2025) Complementary roles of the human perirhinal and the entorhinal cortex in the hippocampal retrieval of episodic memory. _iScience_. https://doi.org/10.1016/j.isci.2025.114241
All scripts are written by J.S. Kim

Please contact me at kimjisun@snu.ac.kr if you have any questions about the codes

The scripts here are based on Matlab r2021b.
